

## Ingredients:
| Ingredients            | Amount |
| ---------------------- | ------ |
| Bourbon| 1.5    |
| Lemon Juice            | .75    |
| Demerara Syrup | .50    |
| 1/2 Egg White          |        |

### Garnish
3 drops Angostura Bitters over foam.

### Instructions:
Combine ingredients in shaker and dry shake.
Add ice.
Shake vigorously for 15-20 seconds
Double Strain into chilled coupe.
Garnish with Bitters
